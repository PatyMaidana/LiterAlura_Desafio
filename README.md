# Desafio LiterAlura

Este projeto faz parte do desafio de Java da Alura. Ele consiste em uma aplicação Java Spring Boot JPA, que faz consulta a API externa https://gutendex.com/ que traz informações de livros e seus autores. 

## Funcionalidades
Esta aplicação permite a consulta de livros na API externa e insere os dados em um banco PostgreSQL, permite demais consultas como por autor, idioma e período em que o autor viveu.

### Instruções 
Para utilizar essa aplicação, faça o download do projeto em seu computador e depois crie um banco de dados no PostgreSQL com o nome LiterAlura e configure suas variáveis de ambiente conforme suas preferências.
